# Expense Tracker
Android App to track Expenses

- Kotlin
- Compose
- MVVM
- Clean Architecture (app, data, domain)
- Navigation (1 Activity -> Multiple Fragments using a navGraph)
- Coroutines, Flows
- StateFlows
- Room PL
- SharedPreferences
